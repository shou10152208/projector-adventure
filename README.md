# 星守の夜 — STARFALL GUARDIANS

> プロジェクターで壁に投影し、Webカメラの前で**全身を動かして**遊ぶ協力型MRゲーム。
> 夜空の守護者となり、降り注ぐ流星・侵略ドローン・巨大ボス「星喰イ」から街を護れ。

複数人が壁の前に並び、**手を振って流星をはじき**、**両手を打ち合わせて衝撃波**を放ち、
**両手を高く上げて必殺技「星の雨」**を発動します。カメラが無い環境では自動でマウス／キーボードに切り替わります。

---

## クイックスタート

> **Python 3** が必要です（多くのPCに標準搭載。無ければ <https://www.python.org/> から）。
> Node.js やビルド作業は **不要**。MediaPipe一式は同梱済みで**完全オフライン**で動きます。

### Windows
1. `start.bat` をダブルクリック
2. 既定のブラウザで自動的に開きます
3. カメラの使用許可を「許可」する

### macOS / Linux / WSL
```bash
./start.sh
```
（または `python3 server.py`）

開かない場合はブラウザで **http://localhost:8000/** を開いてください。

> **重要**: `index.html` をファイルとして直接開くと、ブラウザの制限でカメラとモジュール読込が動きません。
> 必ず上記サーバー経由（`http://localhost…`）で開いてください。

## 同じWiFiの別端末から遊ぶ（LAN）

サーバーは `0.0.0.0` で待ち受けるので、同一WiFiの別端末（スマホ・タブレット・別PC）からアクセスできます。

**まず知っておくこと:**
- **マウス/タッチ操作**なら普通の **HTTP** でOK。
- 別端末で**その端末のカメラ（全身操作）を使う**には **HTTPS が必須**です（ブラウザの仕様で、`http://<IP>` ではカメラが許可されません）。→ `HTTPS=1` で起動。
- **WSL で動かしている場合**は、WSL が NAT のため Windows 側でポート転送が必要です（下記）。

### 手順（WSL の場合）
1. **WSL でサーバーを起動**
   ```bash
   ./start.sh            # マウス/タッチで遊ぶなら
   HTTPS=1 ./start.sh    # 別端末でもカメラを使うなら（自己署名証明書を自動生成）
   ```
2. **Windows 側でポート転送**（管理者 PowerShell で1回）
   ```powershell
   powershell -ExecutionPolicy Bypass -File lan-setup.ps1
   ```
   実行すると別端末用の URL（例 `http://192.168.0.224:8000/`）が表示されます。解除は `lan-setup.ps1 -Remove`。
3. **別端末のブラウザでその URL を開く**
   - HTTP の場合 → マウス/タッチで操作。
   - HTTPS の場合 → 「保護されていない接続」の警告が出るので **「詳細設定 → アクセスする」** で進む（自己署名のため。LAN内なので安全）。その後カメラを「許可」。

> **もっと簡単にしたい場合（Windows 11）**: `%UserProfile%\.wslconfig` に
> ```
> [wsl2]
> networkingMode=mirrored
> ```
> を書いて `wsl --shutdown` → 再起動すると、WSL が Windows と同じLANに直結し、`lan-setup.ps1` 不要で別端末から届きます。

### Windows ネイティブの Python で動かす場合
WSL を使わず `start.bat`（Windows の Python）で起動すれば NAT 問題は無く、`lan-setup.ps1` も不要です。初回のファイアウォール・ダイアログで「アクセスを許可」するだけ。

> セキュリティ: これは LAN内の誰でもアクセスできる開発用サーバーです。公共WiFiでの使用は避けてください。`certs/` の証明書は自動生成物です。

## 遊び方

### カメラモード（全身で操作）
| 動作 | 効果 |
|---|---|
| 手を振る・払う | 手のオーラで流星をはじき飛ばす。**速く振るほど強い斬撃** |
| 両手を打ち合わせる（拍手） | **衝撃波**が広がり、周囲の敵をまとめて吹き飛ばす |
| 両手を頭より高く上げる | ゲージ満タン時に必殺技 **星の雨** を発動（画面の敵を一掃） |
| 体を動かして手を運ぶ | 複数人同時プレイ可能。各自が色違いの守護者になる |

- 画面の前に立つと、自分の輪郭がうっすら映り、手から光のオーラが出ます。
- **タイトルで両手を上げるとゲーム開始**。やられても両手を上げれば再挑戦。

### マウス／キーボードモード（カメラが無い時：自動切替）
| 操作 | 効果 |
|---|---|
| マウス移動 | 手のオーラを動かす |
| 左ボタン長押し | 斬撃を強化（はじける） |
| 右クリック / `C` | 衝撃波 |
| `Space` | 星の雨（ゲージ満タン時）／開始・再挑戦 |
| 矢印 or `WASD` | 手をキーで移動（マウス無し環境向け） |

### 共通キー
`K` キーボードモード ／ `G` カメラ再試行 ／ `M` 消音 ／ `D` デバッグ情報 ／ `F` 全画面

## ゲームの流れ
1. **第一夜（ウェーブ1〜5）**: 流星 → ドローン → 分裂体、と敵が増えていきます。手を振って街を護りましょう。
2. **ウェーブ6 — 星喰イ**: 画面上部に巨大ボスが出現。**光る核（弱点）を狙う**と大ダメージ。
   3つのフェーズを削り切ると…夜はまだ終わらない。
3. **第二夜（ウェーブ7〜11）**: 月が紅く染まり、新たな敵が現れます。
   - 🛡️ **よろい岩** — 触れているだけではほぼ削れない。**素早い斬撃・拍手・爆発**で割る
   - 💣 **バクダン星** — 倒すと大爆発して**周囲の敵を巻き込む**。連鎖が爽快（街に落とすと大ダメージなので注意）
4. **ウェーブ12 — 月喰イ**: 星喰イの真の姿。4フェーズ+全方位ショットの強化ボス。倒せば **夜明け（勝利）**。
5. 街のHPが尽きると陥落（ゲームオーバー）。コンボを繋いでハイスコアを狙え。

### ⭐ 星のかけら
敵を倒すとときどき光る星のかけらが落ちてきます。**手で触れて回収**すると
必殺ゲージが増え、街のHPも少し回復します（消える前にキャッチ！）。

## プロジェクター設置のコツ
- **カメラはプレイヤー側（壁の前に立つ人）に向ける**。プレイヤーは投影された自分を鏡のように見て操作します。
- 部屋はやや暗めに。投影が見やすく、姿勢認識も安定します。
- 全身（特に**両手と頭**）がカメラに写る距離に立ってください。両手上げ・拍手の判定に必要です。
- 起動後 `F` キーでブラウザを全画面にすると没入感が上がります。

## 調整（チューニング）
ゲームバランスは [`src/config.js`](src/config.js) に集約しています。コード変更後はブラウザを再読込するだけ。
- `input.maxPlayers` … 同時認識人数（既定4）
- `input.modelFile` … `pose_landmarker_lite.task`（軽量・既定） / `pose_landmarker_full.task`（高精度）
- `input.delegate` … `'GPU'`（既定） / `'CPU'`（GPUが不安定な時）
- `combat.*` … 斬撃ダメージや拍手の威力
- `city.maxHp`, `enemies.*`, `boss.*` … 難易度

---

## うまく動かない時
- **カメラが映らない / 認識しない**: ブラウザのカメラ許可を確認。`G` で再試行。だめでも自動でマウス操作に切替わります。
- **画面が真っ暗 / 起動しない**: `http://localhost:8000/` でアクセスしているか確認。`F12` のコンソールにエラーが出ていないか確認。
- **動作が重い**: `src/config.js` で `input.maxPlayers` を下げる、`delegate` を `'CPU'`↔`'GPU'` 切替、`render.maxParticles` を下げる。
- **ポート競合**: `PORT=8080 python3 server.py` のように別ポートで起動。
- 推奨ブラウザ: **最新の Google Chrome / Microsoft Edge**（WebGL・getUserMedia 対応）。

## 技術構成
- **言語**: バニラ JavaScript（ES Modules、ビルド不要）
- **姿勢認識**: [MediaPipe Tasks Vision — Pose Landmarker](https://developers.google.com/mediapipe)（複数人・全身33点）。`vendor/` に同梱しオフライン動作。
- **描画**: HTML5 Canvas 2D（加算合成によるネオン発光表現）
- **音**: Web Audio API による完全プロシージャル生成（音源ファイル無し）
- **カメラ**: `getUserMedia` + `<video>`
- **配信**: 付属の `server.py`（`.mjs`/`.wasm` の MIME とクロスオリジン分離ヘッダを設定）

詳細な構成は [`CLAUDE.md`](CLAUDE.md) を参照。
